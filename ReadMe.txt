How to install?
1.Import the database file 'Pr/setup/import.sql'
2.Config the DEFINE.php  'Pr/setup/DEFINE.php'
3.Complete!

Notice:
We use Gettext to support multi language,so this site can auto choose language on the basis of client(default is English)
if you want to add more language please read this:
1.Add the dir which should named the language you want add in path:"Pr/locale/".
2.Add the sub dir named 'LC_MESSAGE' .
3.Generate the binary file '*.mo'(we highly recommend you use Poedit)
4.Complete!
